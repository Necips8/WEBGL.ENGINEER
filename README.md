# WEBGL.ENGINEER - Mein Portfolio als Creative Developer & Senior Backend Expert

Besuchen Sie die Live-Seite hier: [https://necips8.github.io/WEBGL.ENGINEER](https://necips8.github.io/WEBGL.ENGINEER)

## Überblick

Dieses Projekt ist die professionelle Landingpage mit meinem Portfolio, einem Senior Backend Developer und Creative Tech Artist. Es präsentiert eine einzigartige Fusion aus High-End-Backend-Engineering und immersiven 3D-Frontend-Erlebnissen.

Die Website ist als interaktive 3D-Reise konzipiert, die es Besuchern ermöglicht, meine Expertise durch eine Parallax-Scrolling-Umgebung zu erkunden.

## Hauptmerkmale

-   **Immersive 3D-Umgebung:** Erstellt mit **Babylon.js**, bietet einen flüssigen und visuell beeindruckenden Hintergrund.
-   **Creative Coding Galerie:** Eine interaktive Auswahl an WebGL/Shader-Experimenten, darunter:
    -   **Boids & Schwarmintelligenz:** GPU-beschleunigte Simulationen.
    -   **Algorithmische Kunst:** Marching Cubes und prozedurale, auf Rauschen basierende Formen.
    -   **Physik-Simulationen:** Slime-Simulationen und auf Brownschem Molekularbewegung basierende Meereswellen.
-   **KI-unterstützte Entwicklung (AI-Assisted Development):** Eine Kernphilosophie, die zeigt, wie die Integration von KI-Tools die Entwicklungseffizienz um das bis zu 10-fache steigern kann, während gleichzeitig eine hohe Codequalität gewahrt bleibt.
-   **Full-Stack-Expertise:** Unterstreicht einen tiefgreifenden Hintergrund in PHP, Shopware 6 und Symfony.

## Technischer Stack

-   **Frontend:** JavaScript/TypeScript, Babylon.js (WebGL), Vue.js (Admin UI).
-   **Backend:** PHP 8, Shopware 6, Symfony Framework, Doctrine ORM.
-   **Infrastruktur:** Docker, Git, CI/CD (Azure Pipelines).
-   **Methodik:** KI-unterstützte Entwicklung, pragmatische Problemlösung, Remote-First-Workflow.

## Projektstruktur

-   `index.html`: Der Haupteinstiegspunkt mit der Babylon.js 3D-Szene und den UI-Layern.
-   `examples/`: Eine Sammlung von eigenständigen WebGL/Babylon.js Experimenten, die in der Portfolio-Galerie verwendet werden.
-   `data/profil.md`: Zentrale Datenquelle für Berufserfahrung und Qualifikationen.
-   `plan.md`: Die konzeptionelle Roadmap und Designstrategie für das Projekt.
